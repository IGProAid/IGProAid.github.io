DISCRETE STRUCTURE

George Cantor - introduced the concept of Sets.

Set - an unordered collection of objects.


Representation of a Set

1. Roster or Tabular Form - A = {a, e, i, o, u}

2. Set Builder Notation - A = {x:p(x)}


Types of Set

Finite Set - elements are countable. (A = (1, 2, 3))

Infinite Set - elements are uncountable. (A = (1, 2, 3,...))

Singleton Set - has only one element.

Empty Set - set with no element; represented as {} or 0.


Common Sets:

N - Natural Numbers

Z - Integers

Z+ - Positive Integers

Q - Rational Numbers

R - Real Numbers

W - Whole Numbers


Cardinality of a Set - number of elements in a set.

Subset - if every element of A is also an element of B, then B is subset of A.

Union Set - set that contains elements that either in A or in B, denoted as U.

Intersection Set - set containing elements in both A and B.

Disjoint - if intersection of two elements is empty.


Proposition - declarative sentence that is answerable by T or F but not both.

Connectives:

Conjunction - T when both p and q are True

Disconjunction - F when both p and q are False

Negation - opposite answer

Conditional - F when p is True and q is False

Biconditional - T when both p and q have the same Truth Value

Exclusive - T when either p and q is True


Tautology - a formula in Truth Table in which the answers are always True

Contradiction - a formula in Truth Table in which the answers are always False

Contingency - a formula in Truth Table in which the answers are both True and False


Data Structure and Algorithm - systematic way to organize data in order to use it efficiently.
